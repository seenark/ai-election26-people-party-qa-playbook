คุณภาพชีวิต

91

นโยบาย

สุขภาพดีในสิ่งแวดล้อมที่ปลอดภัย มีที่ดินมั่นคงและสวัสดิการที่ทั่วถึง ยกระดับรายได้เกษตรกรและจัดการน้ำอย่างเป็นระบบ สร้างสังคมแห่งการเรียนรู้ที่โอบรับความหลากหลายและเคารพศักดิ์ศรีความเป็นมนุษย์ ท่านสามารถเข้าไปอ่านชุดนโยบายของพรรคประชาชนได้ที่นี่
