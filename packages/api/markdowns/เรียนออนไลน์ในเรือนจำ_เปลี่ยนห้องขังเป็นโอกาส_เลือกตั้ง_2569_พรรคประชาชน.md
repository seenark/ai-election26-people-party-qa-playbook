ทำไมต้องแก้ปัญหา (WHY)
----------------------

1.  **การศึกษาในเรือนจำไม่ทั่วถึงและจำกัด** ระบบการศึกษาในเรือนจำปัจจุบันจำกัดอยู่เฉพาะหลักสูตรพื้นฐาน หรือโครงการเฉพาะจากสถาบันบางแห่ง (เช่น กศน. หรือมหาวิทยาลัยบางแห่ง) เท่านั้น ทำให้ผู้ต้องขังจำนวนมาก ขาดโอกาสในการพัฒนาทักษะอาชีพ หรือเรียนต่อในสาขาใหม่ ๆ อย่างทั่วถึง
2.  **ทักษะผู้ต้องขังไม่สอดคล้องกับตลาดแรงงานยุคใหม่** สังคมภายนอกกำลังก้าวเข้าสู่เศรษฐกิจดิจิทัล และตลาดแรงงานต้องการแรงงานที่มีทักษะใหม่ (Reskill / Upskill) โดยเฉพาะด้านเทคโนโลยีสารสนเทศ การตลาดออนไลน์ การออกแบบ และทักษะภาษา ผู้ต้องขังที่ขาดโอกาสในการเรียนรู้ทักษะเหล่านี้มักประสบปัญหา ไม่สามารถหางานได้เมื่อพ้นโทษ หรือมีโอกาสกลับไปอยู่ในวงจรเดิม นำไปสู่การ **“กระทำผิดซ้ำ”** สูง

เราจะทำอะไร (WHAT)
------------------

เพื่อเป็นการส่งเสริมการลดการกระทำผิดซ้ำ พรรคประชาชนจะให้เครื่องมือแก่ผู้ต้องขังในการซ่อมและสร้างทุนมนุษย์ เพื่อทางเลือกใหม่ ๆ ในการดำเนินชีวิตเมื่อกลับคืนสู่สังคมได้อย่างมีคุณภาพ โดยมีข้อเสนอหลักดังนี้:

1.  1\. **จัดตั้งห้องเรียนออนไลน์ในเรือนจำ**
    *   สร้าง ห้องเรียนออนไลน์เพื่อคืนคนสู่สังคมในเรือนจำทุกแห่ง โดยจัดหาอุปกรณ์ที่เพียงพอต่ออัตราผู้ต้องขังในแต่ละเรือนจำ
2.  2. **แพลตฟอร์มที่เชื่อมการเรียนรู้ที่หลากหลาย** 
    *   จัดทำ แพลตฟอร์มการเรียนออนไลน์ของประเทศ ที่เชื่อมต่อระบบ e-Learning ของกรมราชทัณฑ์ (ซึ่งมีระบบความปลอดภัย) เข้ากับ Thai MOOC ภายใต้กระทรวง อว. เพื่อให้ผู้ต้องขังเข้าถึงหลักสูตรจากมหาวิทยาลัย และ กศน. ที่หลากหลายขึ้น
3.  3\. **หลักสูตรที่ตอบโจทย์ตลาดแรงงาน**
    *   หลักสูตรจะไม่จำกัดเฉพาะการศึกษาตามวุฒิ แต่จะครอบคลุม 3 กลุ่มใหญ่ ได้แก่:
    *   การศึกษาขั้นพื้นฐาน: การเรียนต่อ กศน. หรือสถาบันการศึกษาที่ร่วมทำ MOU
    *   การศึกษาทักษะอาชีพและเศรษฐกิจดิจิทัล: เช่น ออกแบบกราฟิกดิจิทัล, การขายออนไลน์, การตลาดดิจิทัล, การใช้โปรแกรมพื้นฐาน, การเขียน Coding เป็นต้น
    *   การศึกษาการพัฒนาทักษะชีวิตและ Soft Skills: เช่น การบริหารการเงิน, การบริหารเวลา, การสื่อสาร, การจัดการความเครียด, และการวางแผนชีวิตหลังพ้นโทษ

ทำอย่างไรให้สำเร็จ (HOW)
------------------------

1.  **ปรับปรุงกฎหมายเพื่อรองรับการเรียนรู้ดิจิทัล**
    *   แก้ไขเพิ่มเติม พ.ร.บ. ราชทัณฑ์ พ.ศ. 2560 มาตรา 43 (สิทธิในการได้รับการศึกษา) โดยเพิ่มข้อความ “การเรียนรู้ผ่านสื่ออิเล็กทรอนิกส์และเทคโนโลยีสารสนเทศภายใต้การควบคุมของเจ้าหน้าที่” เพื่อให้เรือนจำสามารถใช้เครื่องมืออิเล็กทรอนิกส์ได้อย่างถูกต้องตามกฎหมาย
2.  **บูรณาการความร่วมมือระหว่างหน่วยงาน**
    *   จัดให้มีแพลตฟอร์มออนไลน์ และออกแบบหลักสูตรร่วมกับเครือข่ายสถาบันการศึกษา รวมถึงออกใบรับรอง โดย Thai MOOC สังกัด กระทรวง อว.
    *   พัฒนาและรับรองการเรียนเพื่อ เทียบโอนผลการเรียนสู่การจ้างงานจริง โดย สถาบันคุณวุฒิวิชาชีพ
3.  **อำนวยความสะดวกในการจัดการเรียนการสอนออนไลน์**
    *   สร้างห้องเรียนออนไลน์ และจัดซื้ออุปกรณ์อิเล็กทรอนิกส์ (โทรทัศน์ และคอมพิวเตอร์) ของส่วนกลางในแต่ละเรือนจำให้เพียงพอ
    *   ระบบการเรียนออนไลน์ต้องปลอดภัย โดยไม่เปิดให้เชื่อมต่ออินเทอร์เน็ตภายนอก เพื่อรักษาความปลอดภัยของข้อมูลและการควบคุม
4.  **พัฒนาระบบ e-Learning ของกรมราชทัณฑ์**
    *   พัฒนาและขยายระบบ e-Learning เดิมของกรมราชทัณฑ์ให้ เชื่อมโยงกับแพลตฟอร์มกลางของประเทศ
    *   จัดทำ ระบบล็อกอินเฉพาะรายบุคคล เพื่อเก็บข้อมูลการเรียนรู้ของผู้ต้องขังแบบต่อเนื่อง
    *   ยกระดับระบบ e-Learning เดิมให้สามารถรองรับวิดีโอสตรีมมิ่ง, การทดสอบออนไลน์ และระบบสะสมผลการเรียน (Credit Bank) เป็นต้น
