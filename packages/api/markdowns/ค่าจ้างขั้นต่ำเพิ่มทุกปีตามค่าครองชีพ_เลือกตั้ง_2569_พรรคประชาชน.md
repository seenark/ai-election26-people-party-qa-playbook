ทำไมต้องแก้ปัญหา (WHY)
----------------------

1.  **สูตรคำนวณที่ไม่สะท้อนความจริง:** สูตรคำนวณที่ใช้ในการปรับขึ้นอัตราค่าจ้างขั้นต่ำในปัจจุบัน ส่งผลลดทอนสัดส่วนผลตอบแทนที่แรงงานควรได้รับตามผลิตภาพแรงงาน ทำให้คนทำงานไม่ได้รับผลประโยชน์ที่ยุติธรรมจากการเติบโตของเศรษฐกิจอย่างที่ควรจะเป็น
    
2.  **ความล่าช้าของกลไกคณะกรรมการ**: แม้เศรษฐกิจจะเติบโตหรือเงินเฟ้อจะพุ่งสูง แต่กฎหมายเปิดช่องให้คณะกรรมการค่าจ้าง "ตกลงกันเอง" ว่าจะขึ้นค่าจ้างขั้นต่ำหรือไม่ก็ได้ ทำให้การปรับค่าจ้างไม่ทันต่อค่าครองชีพที่เพิ่มขึ้น
    
3.  **แนวคิดแรงงานราคาถูก**: ประเทศไทยติดอยู่กับกับดักการใช้ "ค่าแรงต่ำ" เป็นจุดขายเพื่อดึงดูดการลงทุน จนละเลยแนวคิดที่ว่าแรงงานควรมีรายได้เพียงพอสำหรับดูแลสมาชิกในครอบครัวอย่างน้อย 2 คนให้มีคุณภาพชีวิตที่ดี ไม่ใช่เพียงเพื่อประทังชีวิตวันต่อวัน
    

เราจะทำอะไร (WHAT)
------------------

เราเสนอให้มีการปรับสูตรการคำนวณค่าจ้างขั้นต่ำเพื่อคุณภาพชีวิตที่ดีของแรงงานไทย มุ่งสู่เป้าหมายค่าจ้างที่อยู่ได้จริงภายในปี 2579 ดังนี้:

1.  **มุ่งสู่เป้าหมาย “ค่าจ้างที่อยู่ได้จริง” ภายในปี 2579**: ตั้งเป้าหมายและกำหนดแผนงานเพื่อให้แรงงานไทยได้รับค่าจ้างขั้นต่ำที่เพียงพอต่อการดูแลตนเองและครอบครัวให้มีคุณภาพชีวิตที่ดีได้ภายในปี 2579
    
2.  **เปลี่ยนสูตรคำนวณให้เป็นธรรม**: ปรับสูตรการคำนวณค่าจ้างขั้นต่ำให้เป็นธรรมกับแรงงาน โดยพิจารณาถึงอัตราเพิ่มขึ้นของอัตราสมทบของแรงงาน ผลิตภาพแรงงาน และอัตราเงินเฟ้อเป็นสำคัญ
    
3.  **ปฏิรูปคณะกรรมการค่าจ้าง (สัดส่วนภาครัฐ)**: ปรับเหตุแห่งการพ้นจากตำแหน่งของคณะกรรมการค่าจ้าง โดยให้ กรรมการค่าจ้างเฉพาะสัดส่วนผู้แทนฝ่ายรัฐบาล พ้นจากตำแหน่งเมื่อคณะรัฐมนตรีที่แต่งตั้งกรรมการผู้นั้นพ้นจากตำแหน่ง เพื่อให้การผลักดันนโยบายค่าจ้างยึดโยงกับนโยบายของคณะรัฐมนตรีที่เข้ามาบริหารประเทศ
    

ทำอย่างไรให้สำเร็จ (HOW)
------------------------

แก้ไข พ.ร.บ. คุ้มครองแรงงาน ดังนี้

*   **มาตรา 80-81** กำหนดให้กรรมการค่าจ้างสัดส่วนรัฐบาล พ้นจากตำแหน่งเมื่อคณะรัฐมนตรีที่แต่งตั้งกรรมการผู้นั้นพ้นจากตำแหน่ง
    
*   **มาตรา 87** ให้อำนาจรัฐมนตรีออกประกาศเพื่อกำหนดสูตรในการคำนวณอัตราการขึ้นค่าจ้างขั้นต่ำ ใช้เป็นบรรทัดฐานเดียวกันทั่วประเทศ 
    
*   **มาตรา 88** กำหนดให้คณะกรรมการค่าจ้างต้องประชุมเพื่อพิจารณาปรับขึ้นค่าจ้างอย่างน้อยปีละหนึ่งครั้ง โดยใช้ผลลัพธ์จากสูตรที่คำนวณได้เป็นหลักในการเจรจา แต่จะมีมติให้ "ลด" หรือ “ไม่ขึ้น” ค่าจ้างขั้นต่ำไม่ได้
