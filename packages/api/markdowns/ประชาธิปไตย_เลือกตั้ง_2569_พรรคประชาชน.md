ประชาธิปไตย

7

นโยบาย

มุ่งสร้างรัฐธรรมนูญฉบับประชาชน เพิ่มการมีส่วนร่วมของประชาชนโดยตรงกับสถาบันการเมืองและโครงการรัฐที่มีผลกระทบในพื้นที่ คุ้มครองสิทธิเสรีภาพจากการฟ้องปิดปาก (Anti-SLAPPs) หยุดวัฒนธรรมลอยนวลพ้นผิด และเดินหน้านิรโทษกรรมประชาชนเพื่อคืนความหวังให้ระบอบประชาธิปไตย ท่านสามารถเข้าไปอ่านชุดนโยบายของพรรคประชาชนได้ที่นี่
